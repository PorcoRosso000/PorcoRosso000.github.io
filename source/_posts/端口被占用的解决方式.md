查看所有的端口占用情况cmd 

输入命令：

```
netstat -ano
```

查看指定端口的占用情况

```
netstat -aon|findstr "8080"
```

查看PID对应的进程

```
tasklist|findstr "9524"
```

结束该进程

```
taskkill /f /t /im java.exe
```
